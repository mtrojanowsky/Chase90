💡 Wskazówki — jak układać tydzień 7 zasad
▾
Siłowy nigdy przed akcentem. Min. 1 dzień przerwy przed biegiem mocnym. Zmęczone mięśnie stabilizujące = ryzyko kontuzji.
Long run = dzień po wolnym lub easy. Nogi muszą wejść w długi bieg możliwie wypoczęte.
Mobility po biegu, nie przed akcentem. Przed akcentem tylko krótka rozgrzewka (10 min).
Suplementy codziennie. Omega D3+K2 i kolagen — rano. Magnez — wieczorem. Kreatyna i białko — do 60 min po treningu.
Kolagen działa tylko z vit. C. Popijaj sokiem lub weź osobny suplement vit. C razem z nim.
Między akcentem a long runem — bufor. Min. 1–2 dni łatwego biegania lub odpoczynku.
RPE jako kompas. Easy: RPE 3–4. Akcent: RPE 7–8. Long run końcówka: RPE 6–7.
