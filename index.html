💡 Wskazówki — jak układać tydzień 7 zasad
▾
Siłowy nigdy przed akcentem. Dzień siłowy stawiaj zawsze min. 1 dzień przed bieganiem mocnym (interwały, tempo). Zmęczone mięśnie stabilizujące = ryzyko kontuzji.
Long run = dzień po wolnym lub easy. Najlepiej niedziela po sobocie bez biegu. Nogi muszą wejść w długi bieg możliwie wypoczęte.
Mobility po biegu, nie przed akcentem. Sesja mobility/wzmacniająca w dniu wolnym lub po easy runie. Przed akcentem tylko krótka rozgrzewka (10 min).
Suplementy codziennie. Omega D3+K2 i kolagen — rano z posiłkiem lub na czczo + vit. C. Magnez — wieczorem. Kreatyna i białko — w ciągu 60 min po każdym treningu.
Kolagen działa tylko z vit. C. Sam kolagen bez witaminy C wchłania się słabo. Popijaj sokiem pomarańczowym lub weź osobny suplement vit. C razem z nim.
Między akcentem a long runem — bufor. Minimum 1–2 dni łatwego biegania lub odpoczynku pomiędzy mocnym treningiem a długim biegiem.
RPE jako kompas. Easy run powinien być RPE 3–4. Akcent/interwały: RPE 7–8. Long run z zadaniami: RPE 6–7 na końcowych kilometrach. Jeśli wyżej — coś nie gra z regeneracją.
